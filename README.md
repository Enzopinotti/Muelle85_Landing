# Muelle85 — Landing inmersiva para una marca de campers premium

### 🗓️ Fecha: 2025-12-10
### 📂 Categoría: Frontend
### 🛠️ Tecnologías: React, Vite, JavaScript, SCSS, GSAP, ScrollTrigger, Responsive Design, Media Optimization, 3D Asset Integration

---

## 📝 Descripción
Landing inmersiva para Muelle85, una marca de campers premium, con animaciones GSAP ligadas al scroll, video optimizado y un render 3D interactivo del producto.

## 🚀 Detalle del Proyecto
Desarrollé la landing de **Muelle85**, una marca de campers y trailers premium concebida como una propuesta de estilo de vida, aventura e identidad. El objetivo fue construir una experiencia web de alto impacto visual que transmitiera robustez, diseño e ingeniería, combinando narrativa emocional, una estructura clara de marca y llamadas a la acción orientadas a conversión.

A nivel técnico, trabajé en una interfaz moderna, responsive y optimizada para contenido visual pesado, integrando animaciones con **GSAP** y **ScrollTrigger** para potenciar el recorrido de usuario. La experiencia incluye transiciones ligadas al scroll, una sección donde el video acompaña el desplazamiento para reforzar la narrativa del producto y un render 3D de un camper que rota con el scroll, aportando una sensación más inmersiva y diferencial dentro de la landing.

Además, se optimizaron los recursos multimedia para lograr un equilibrio entre impacto visual y rendimiento, reduciendo el peso de videos y assets sin comprometer la percepción premium del sitio. El resultado fue una pieza digital pensada no solo para presentar un producto, sino para posicionar una marca con personalidad, comunidad y una visión aspiracional del viaje.
